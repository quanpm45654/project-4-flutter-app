# project_4_flutter_app

A Flutter project.

## Getting Started

### Prerequisites

- Dart SDK version ^3.8.0
- Flutter SDK version ^3.32.0
- Android SDK
- Android NDK

### Run Locally

Install dependencies:

```bash
  flutter pub get
```

Run the app in debug mode:

```bash
  flutter run lib/main.dart
```

Run the app in release mode:

```bash
  flutter run --release lib/main.dart
```
